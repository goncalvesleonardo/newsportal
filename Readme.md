# News Portal

## Descrição do Projeto <p>Projeto criado para listar noticias da Api do New York Times</p>

### Tecnologias utilizadas:
* React Navigation
* Styled Components
* Axios
* Eslint
* Prettier
* React Hooks (useState, useEffect)

 #### Para executar o projeto rodar os comandos:
* yarn
* react-native run-android
